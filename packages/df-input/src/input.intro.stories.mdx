# Input - Get started

Use inputs to allow users to enter, select and search for text.

## Install

Add Input to your project.

`npm install --save @df/input`

### **Use**

`import Input from '@df/input';`

```
  const example = () => (
    <div>
       <Input value={'Label'} />
    </div>
)
```
