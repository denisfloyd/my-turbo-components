# Button - Get started

Labels are right-aligned, succinct and descriptive. Use sentence case.

If a long label is needed, it should wrap and not be truncated.

## Install

Add Label to your project.

`npm install --save @tempo-io/tui-label;`

### **Use**

`import Label from '@tempo-io/tui-label';`

and use them like so

```
  const example = () => (
    <Button>Button Text</Button>
  )
```
